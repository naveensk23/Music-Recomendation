Step 1: Create Spotify API ------------------------------>  https://developer.spotify.com/dashboard/login

Step 2: Create youtube API on GCP console

Step 3: conda create -n env_name python==3.6.9

Step 4: conda activate env_name

Step 5: pip install -r requirements.txt

Step 6: export YOUTUBE_API_KEY="API key"

Step 7: export SPOTIFY_CLIENT_ID="API key"

Step 8: export SPOTIFY_CLIENT_SECRET="API key"

Step 9: RUN : python app.py
